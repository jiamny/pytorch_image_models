# Train image models with pytorch c++

Train torch image models on the CIFAR10 data.


## Prerequisites
- cmake   3.18+
- PyTorch 2.0.0
- opencv  4.5.4/4.7.0
- CUDA    11.7
- cudnn   8.7
- ubuntu 22.04
- gcc-11/g++-11 for C++ 17
